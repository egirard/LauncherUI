# Technology Stack

This project uses the following core technologies.

## Core
- **Language**: TypeScript
- **Framework**: SvelteKit (Svelte 4)
- **Build Tool**: Vite
- **Backend**: Firebase (Auth + Firestore)

## Testing
- **Unit Testing**: Vitest
- **E2E Testing**: Playwright (zero-pixel tolerance)

## Linting & Formatting
- **Biome**: Linting and formatting
